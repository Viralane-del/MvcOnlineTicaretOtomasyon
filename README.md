# MVCOnlineTicariOtomasyon

Bir çevrimiçi ticari otomasyon sistemi. Ürün yönetimi, siparişler, fatura ve stok takibi gibi işlevler içerir.

---

## Özellikler

- Kullanıcı ve rol bazlı giriş  
- Ürün, kategori ve stok yönetimi  
- Sipariş ve fatura sistemi  
- Dashboard & raporlama  
- Responsive tasarım  
- CRUD işlemleri

---

## Teknolojiler

- **Backend:** ASP.NET Core MVC  
- **Veritabanı:** SQL Server, EF Core  
- **Frontend:** HTML, Bootstrap, JavaScript  
- **Diğer:** Git
